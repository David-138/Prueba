# Sumo Balls

## Instalación
Para instalar este maravilloso juego hay que abrir Git Bash, ir al directorio donde quieras el juego y ejecutar el comando `git clone https://github.com/David-138/Prueba.git`. Tras esto hay que abrirlo en Unity y desde ahí, o bien ejecutarlo desde el propio editor de Unity o generar el juego.
